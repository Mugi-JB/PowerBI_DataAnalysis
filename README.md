# Data Analysis using Power BI and SQL
In this project, I follow along [Ali Ahmad's youtube playlist on Data Analyst Portfolio Project](https://youtu.be/z7o5Wju-PZg). My intentions are to explore, learn and practice data analysis using Power BI and SQL and in the end put together a complete project to show for what I have learnt and what I am cpable of in terms of data analytics.

## Tools:
- [Power BI Desktop](https://powerbi.microsoft.com/en-us/desktop/)

- [SQL Server Management Studio](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?redirectedfrom=MSDN&view=sql-server-ver15)
